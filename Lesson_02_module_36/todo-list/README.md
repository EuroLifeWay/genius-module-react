# React TODO List

A simple and functional todo list app built with React.

## 🚀 Features

- **Adding todos**: Create new todo list items.
- **Editing**: Change todo text by clicking the "Edit" button or
  double-clicking.
- **Delete**: Delete a single todo or clear all completed todos.
- **Complete status**: Mark todos as completed by checking a checkbox or
  clicking on the text (checkbox + strikethrough).
- **Bulk actions**: "Do all" / "Cancel all" buttons.
- **Statistics**: Real-time counter of completed and total todos.
- **Saving**: Automatic list syncing with `localStorage` (data is not lost after
  a restart).

## 🛠 Technologies

- **React** (Hooks: `useState`, `useEffect`, `useRef`).
- **CSS3** (Responsive layout).
- **JavaScript** (ES6+, Array methods: `map`, `filter`).
